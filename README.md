### Prerequisites

- PNPM (is recommended for its efficiency and speed)

  ```sh
  npm install -g pnpm
  ```

- o NPM

  ```sh
  npm install npm@latest -g
  ```

### Install

1. Clone repository

   ```sh
   git clone https://github.com/AlejandroSenior/rock-paper-scissors-astro
   ```

2. Install packages

   ```sh
   pnpm install
   ```

3. Run project

   ```sh
   pnpm run dev
   ```
